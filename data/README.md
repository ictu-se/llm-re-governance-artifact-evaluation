# Data

`tasks/llm_re_tasks_v240.jsonl` contains the 240 benchmark tasks used in the main experiment.

`tasks/llm_re_tasks_v240_ablation48.jsonl` contains the balanced 48-task subset used for the schema-prompt ablation.

Each JSONL row is one task with stakeholder intent, system context, available evidence, disallowed behavior, and expected keyword signals for automatic evaluation.
