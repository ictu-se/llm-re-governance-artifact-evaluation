# Results

`predictions/` contains saved JSONL model outputs used by the current paper draft.

`final/` contains recomputable automatic metrics, rubric scores, model/family summaries, statistical comparisons, and the final prediction manifest.

`ablation/` contains schema/no-schema prompt-ablation predictions and rubric summaries.

Synthetic placeholder human-evaluation files are intentionally excluded. Human-review templates and agreement scripts are provided in `scripts/`.
